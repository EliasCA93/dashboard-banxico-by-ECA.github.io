# mexican-economic-indicators
Site to visualize report analytics from https://github.com/EliasCA93/Mexican_BI

This dashboard pretend's to be an example of advanced data science analytics to extract knowledge from public data.
This is the representation of Data Engineer ETL process, to extract, transform and load data for ML case studies.

Unsupervised learning is use to get insights from public data (SIE API Banxico).

Key features:
* Mexican economic indicators from Banxico.
* Data viz --> BI implementation.
* EDA.
* Unsupervised Learning (dimensionality reduction, and clustering).
